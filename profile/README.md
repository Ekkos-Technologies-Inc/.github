**ekkOS** is a cognitive memory system for AI coding assistants. It gives Claude Code, Cursor, and Windsurf persistent memory across sessions — so your AI remembers what worked, what didn't, and never makes the same mistake twice.

</div>

---

### The Problem

Every AI coding session starts from zero. Your assistant forgets the bugs it fixed, the patterns it learned, and the preferences you set. You repeat yourself constantly.

### The Solution

ekkOS gives AI assistants an **11-layer memory architecture** — from working memory (current session) to collective intelligence (cross-project patterns). Solutions get forged into reusable patterns. Mistakes become anti-patterns. Your AI gets smarter with every session.

### Architecture

</div>

```
┌─────────────────────────────────────────────────────────────┐
│                    ekkOS_ Memory Layers                     │
├──────────┬──────────────────────────────────────────────────┤
│  L1      │  Working Memory        (Redis - current session) │
│  L2      │  Episodic Memory       (conversation history)    │
│  L3      │  Semantic Memory       (embeddings & knowledge)  │
│  L4      │  Pattern Memory        (proven solutions)        │
│  L5      │  Procedural Memory     (step-by-step workflows)  │
│  L6      │  Collective Memory     (community patterns)      │
│  L7      │  Meta Memory           (pattern effectiveness)   │
│  L8      │  Codebase Memory       (project-specific)        │
│  L9      │  Directives            (MUST/NEVER/PREFER/AVOID) │
│  L10     │  Conflict Resolution   (contradiction handling)  │
│  L11     │  Secrets Vault         (AES-256-GCM encrypted)   │
└──────────┴──────────────────────────────────────────────────┘
```
